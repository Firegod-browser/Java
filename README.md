# Java RPG Game

This repository contains a simple Java role‑playing game (RPG) I built as one of my early Java projects.  
It focuses on core RPG concepts like heroes, enemies, items, and turn‑based combat using plain Java classes.

## Overview

The goal of this project was to learn Java by designing a small RPG system from scratch:

- `Hero`, `Enemy`, and `Item` classes model the main pieces of an RPG.
- A `Main` class runs the game loop and manages battles.
- Story and design ideas are supported by external notes and guides.

Everything runs in the console, so the focus is on logic and structure rather than graphics.

## Files and structure

All game files are inside the `Nikhil Deshpande java game coding` folder:

- `Main.java` / `Main.class`  
  Entry point of the game. Sets up the hero, enemies, items, and runs the main game loop or battle sequence.

- `Hero.java` / `Hero.class`  
  Represents the player’s character. Typically includes stats like health, attack power, and methods for actions (for example: attack, take damage, use item).

- `Enemy.java` / `Enemy.class`  
  Represents enemies the hero fights. Similar to `Hero`, but with its own stats and behavior.

- `Item.java` / `Item.class`  
  Represents items used in the game (for example: potions, weapons, or other bonuses), and how they affect the hero or battle.

- `RPG Guide.lnk`  
  A shortcut link to an external design or rules document used while planning the game (might not be directly usable on other systems, but shows planning work).

- `Storyboard.lnk`  
  A shortcut link to the game’s storyboard or narrative outline (also system‑specific, but indicates pre‑planning of story and flow).

- `chat_history.json`  
  A JSON file capturing some of the AI or chat‑based brainstorming used while designing the game.

The `.class` files are compiled versions of the `.java` files and can be regenerated by compiling the code.

## Story and gameplay

This RPG is designed as a console‑based adventure with a simple but structured flow:

- You control a hero with defined stats.
- You encounter enemies and make choices about how to fight them.
- Items can be used to heal, boost stats, or otherwise change the outcome of battles.
- The game progresses through a sequence of turns until the hero or enemy is defeated.

## How to run the game

You’ll need a Java Development Kit (JDK) installed.

1. Install Java  
   - Download a JDK from https://adoptium.net/ or Oracle, and make sure `java` and `javac` are available in your terminal.

2. Clone this repository:
   ```bash
   git clone https://github.com/Nikhil-Deshpande-25/Java.git
   cd Java
   ```

3. Go into the game folder:
   ```bash
   cd "Nikhil Deshpande java game coding"
   ```

4. Compile the Java source files:
   ```bash
   javac Main.java Hero.java Enemy.java Item.java
   ```

5. Run the game:
   ```bash
   java Main
   ```

If you prefer using an IDE (like IntelliJ IDEA, Eclipse, or VS Code with Java support), you can import this folder as a project and run the `Main` class directly from there.
